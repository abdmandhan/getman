# GETMAN APP

## Overview

This is the web application for the Getman platform. It is built with Nuxt.js and Vuetify. an App for storing http request similar to postman

## Objectives

- Create a web application for storing http request similar to postman
- Create a web application for storing http response similar to postman
- Create a web application for storing http request and response similar to postman
- Create a web application for storing http request and response similar to postman

## bypass CORS strategy

- request to API will be sent over the SSR

## Technologies
- Nuxt.js
- Vuetify
- Prisma
- PostgreSQL
- Bun
- Tailwind CSS

## Installation

```bash
bun install
```

## Running the application

```bash
bun run dev
```
